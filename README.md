## Run Locally

Clone the project

```bash
  git clone https://github.com/orpic/delicious-meal/
```

Go to the project directory

```bash
  cd delicious-meal
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
